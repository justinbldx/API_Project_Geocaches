# API_Project_Geocaches
Répertoire Git contenant les fichiers pour le projet de la matière "API Rest"

### Lancement du projet

Lancer le docker
```bash
docker compose up -d
```

Compiler le projet et le lancer
```bash
npm run build && npm run start
```
